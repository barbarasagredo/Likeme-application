# ⌞⌝ LikeMe Application

Aplicación desarrollada como parte del desafío "Like Me (Parte II)" de Desafío Latam. Web de la aplicación [aquí](https://barbarasagredo.github.io/Likeme-application/)!

## 📖 Descripción

La red social Like Me está comenzando su desarrollo y busca centrarse en la interacción de los usuarios a través de posts y likes.

El objetivo de este proyecto fue desarrollar el servidor backend que permita:

- Consultar publicaciones existentes
- Crear nuevos posts
- Almacenar la información en una base de datos
- Agregar conteo de likes
- Eliminar posts

Todo esto utilizando Node.js, Express y PostgreSQL. 

## 🚀 Tecnologías utilizadas

- Node.js
- Express
- PostgreSQL
- pg (node-postgres)
- CORS
- JavaScript
- React + Vite

## ⚙️ Instalación

Clonar el repositorio:
```
git clone https://github.com/barbarasagredo/Likeme-application.git
```
Entrar a la carpeta del proyecto:
```
cd Likeme-application
```
Instalar dependencias:
```
npm install
```
Ejecutar el servidor:
```
node index.js
```
El servidor se ejecutará en: http://localhost:3000
